Manipulação do DOM

O que é DOM?

  • DOM - Document Object Model
  • Estrutura de árvore criada pelo browser para o Javascript
  • Representa o HTML como objetos
  • A partir dele o Javascript se comunica com o HTML

Encontrando elementos

  • .getElementById
    Encontra um elemento através do ID informado.
    EXEMPLO
    HTML
    <span id="getID">EXEMPLO</span>

    Tente você
    document.getElementById("getID")
  • .getElementsByClassName
    Encontra um elemento através da classe informada.
    EXEMPLO
    HTML
    <span class="getClass">EXEMPLO</span>

    Tente você
    document.getElementsByClassName("getClass")

    Retorna um HTML Collection, pois vários elementos podem ter a mesma classe.
    Pode ser manipulado de forma semelhante a um array. Tente você
    let getClass = document.getElementsByClassName("getClass")

    console.log(getClass[0])
  • .getElementsByTagName
    Encontra um elemento através do ID informado.
    EXEMPLO
    HTML
    <mark>EXEMPLO</mark>

    Tente você
    document.getElementsByTagName("mark")

    Da mesma forma que com as classes, também retorna um HTML Collection, pois vários elementos podem ter a mesma tag.
    Pode ser manipulado de forma semelhante a um array. Tente você
    let getTag = document.getElementsByTagName("mark")

    console.log(getTag[0])
  • .querySelector
    Permite usar um seletor da mesma forma em que são usados dentro do CSS, ou seja, usando # para IDs, . para classes e a tag em si para tags. Tente você
    document.querySelector("#getID")

    document.querySelector(".getClass")

    document.querySelector("mark")
  • .querySelectorAll
    Semelhante ao querySelector, mas o querySelector retorna apenas um elemento com aquelas características, enquanto o querySelectorAll retorna todos os elementos com aquelas características.
    Tente você
    document.querySelectorAll("h2")

    Retorna um NodeList, já que podem ser retornados vários elementos.
    Pode ser manipulado de forma semelhante a um array. Tente você
    let selectAll = document.querySelectorAll("h2")

    console.log(selectAll[0])

    console.log(selectAll[1])

Manipulando o conteúdo

  • .textContent
    Mostra o conteúdo textual que está dentro do elemento.
    EXEMPLO
    HTML
    <span class="textContent">EXEMPLO</span>

    Tente você let texto = document.querySelector(".textContent")

    texto.textContent
  • .innerText
    Permite modificar o conteúdo textual do elemento.
    EXEMPLO
    HTML
    <span class="innerText">EXEMPLO</span>

    Tente você
    texto = document.querySelector(".innerText")

    texto.innerText = "TESTE"

    texto.textContent
  • .innerHTML
    Permite modificar o HTML do elemento.
    EXEMPLO
    HTML
    <span class="innerHTML">EXEMPLO</span>

    Tente você
    texto = document.querySelector(".innerHTML")

    texto.innerHTML

    texto.innerHTML = "<h1>TESTE</h1>"

    texto.innerHTML

Alterando estilos

  • .style.property
    Permite modificar a estilização do elemento.
    Os nomes das propriedades do CSS devem ser dadas em camel case.
    EXEMPLO
    HTML
    <span class="styleProperty">EXEMPLO</span>

    Tente você
    let estilo = document.querySelector(".styleProperty")

    estilo.style.backgroundColor = "blue"
  • .classList
    Permite modificar as classes dos elementos.
    • .add()
      Permite adicionar classes ao elemento.
      EXEMPLO
      HTML
      <span class="addClass">EXEMPLO</span>

      Tente você
      let add = document.querySelector(".addClass")

      add.classList.add("topico")
    • .remove()
      Permite remover classes ao elemento.
      EXEMPLO
      HTML
      <span class="outline">EXEMPLO</span>

      Tente você
      let remove = document.querySelector(".outline")

      add.classList.remove("outline")

Criando elementos

  • .createElement()
    Permite criar novos elementos.
    Tente você
    let novoElemento = document.createElement("img")
     Após criado, é possível fazer modificações no elemento. Tente você
    novoElemento.src = "src/images/JavaScript-logo.png"

Inserindo elementos no HTML

  • .append()
    Permite inserir o elemento criado na página após o elemento usado como base.
    Tente você
    document.getElementById("append").append(novoElemento)
     Após ser inserido, ainda assim é possível fazer modificações no elemento.
    Tente você
    novoElemento.width = 250
  • .prepend()
    Permite inserir o elemento criado na página antes do elemento usado como base.
    Tente você
    document.getElementById("prepend").prepend(novoElemento)
